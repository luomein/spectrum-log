# Browser-Intern-URLs Cheatsheet DE

## Microsoft Edge
```plaintext
edge://policy/               # Edge-Richtlinien
edge://settings/             # Einstellungen
edge://downloads/            # Downloads
edge://history/              # Verlauf
edge://favorites/            # Lesezeichen/Favoriten
edge://extensions/           # Erweiterungen verwalten
edge://apps/                 # Apps verwalten
edge://taskmanager/          # Edge Task-Manager
edge://flags/                # Experimentelle Funktionen
edge://about/                # Über-Seite
edge://crashes/              # Absturzberichte
edge://gpu/                  # GPU-Informationen
edge://net-internals/        # Netzwerk-Interna
edge://predictors/           # Voraussage-Dienste Debugging
edge://sync-internals/       # Synchronisations-Interna
edge://webrtc-internals/     # WebRTC-Interna
edge://components/           # Komponenten
edge://settings/content      # Website-Berechtigungen
edge://tracing/              # Tracing
edge://settings/reset        # Einstellungen zurücksetzen
edge://settings/privacy      # Datenschutz- und DNS-Einstellungen
edge://settings/privacy#certificates  # Zertifikatsverwaltung
```
