# Browser Internal URLs Cheatsheet EN

## Microsoft Edge
```plaintext
edge://policy/               # Edge Policies
edge://settings/             # Settings
edge://downloads/            # Downloads
edge://history/              # History
edge://favorites/            # Bookmarks/Favorites
edge://extensions/           # Extensions Management
edge://apps/                 # Apps Management
edge://taskmanager/          # Edge Task Manager
edge://flags/                # Experimental Flags
edge://about/                # About Page
edge://crashes/              # Crash Information
edge://gpu/                  # GPU Information
edge://net-internals/        # Network Internals
edge://predictors/           # Predictive Service Debugging
edge://sync-internals/       # Sync Internals
edge://webrtc-internals/     # WebRTC Internals
edge://components/           # Components
edge://settings/content      # Site Permissions
edge://tracing/              # Tracing
edge://settings/reset        # Reset Settings
edge://settings/privacy      # Privacy and DNS Settings
edge://settings/privacy#certificates  # Certificates Management
```
