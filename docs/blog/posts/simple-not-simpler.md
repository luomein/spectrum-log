# 減法不是化約

簡單不是簡化，有句話常以愛因斯坦名言被引用，“Everything should be made as simple as possible, but no simpler”（此段話本身也是簡化過的，原出處更長）

帆船的生活是減法的生活，一不留意就會看起來像流浪者的生活。

重點是建立生活的原則，區分可簡化與不可簡化，建立生活公約。

如果船主邋遢，別人就會跟著踐踏這艘船。
