# 什麼是迷人的港口小鎮

西村幸夫說：「打動人心的風景同時具備『多樣』與『和諧』的特質」

台灣的觀光景點發展方向大致三類：小吃、文創商品、拍照熱點，這適用於一次性的消費，但若希望熟客回流，則需要回到生活本身，生活本身就能提供源源不絕的談話素材、觀點、五感經驗。這需要把生活連結到溝通，是一種有意識的生活方式。

有意識的生活方式，有幾個練習點：

* 覆盤（replay）

* 換位思考

* 心象練習

迷人的港口小鎮，能演活了「生活」這部戲。與此相對的景點，則只是道具陳列，或由不入戲的演員演出。
